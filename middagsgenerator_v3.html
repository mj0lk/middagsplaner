Familiekode
1
Gå til console.firebase.google.com og logg inn med en Google-konto
2
Klikk "Add project" → gi det et navn (f.eks. "Familiemiddager") → klikk deg gjennom til prosjektet er klart
3
Velg Build → Realtime Database → Create database → velg region europe-west1 → velg "Start in test mode" → klikk Enable
4
Kopier Database URL som vises øverst — den ser omtrent slik ut:
https://ditt-prosjekt-default-rtdb.europe-west1.firebasedatabase.app/
Lim inn Database URL her
⚠️ Firebase test-modus varer i 30 dager. Gå til Rules i Realtime Database og sett reglene permanent til ".read": true, ".write": true for å unngå at tilgangen utløper.
